# Chrome Web Store Asset Checklist

This checklist outlines the visual assets required for the HiNote extension's Chrome Web Store listing.

## Promotional Tiles

- [ ] **440x280 PNG (Small):** `store_assets/promo-tile-small-440x280.png`
- [ ] **920x680 PNG (Marquee):** `store_assets/promo-tile-marquee-920x680.png` (optional, but recommended)

## Screenshots

A minimum of 3 screenshots are required, showcasing the extension's functionality.

- [ ] **1280x800 PNG (Screenshot 1):** `store_assets/screenshot-1-1280x800.png`
  - **Content:** Show the highlighting functionality in action on a webpage.
- [ ] **1280x800 PNG (Screenshot 2):** `store_assets/screenshot-2-1280x800.png`
  - **Content:** Show the side panel with a list of notes for the current page.
- [ ] **1280x800 PNG (Screenshot 3):** `store_assets/screenshot-3-1280x800.png`
  - **Content:** Show the global search feature with results from multiple pages.

**Note:** The placeholder images in the `store_assets` directory are simple gray boxes with the dimensions written on them. These should be replaced with actual designs.
